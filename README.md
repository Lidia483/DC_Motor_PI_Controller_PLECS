# DC_Motor_PI_Controller_PLECS
DC Motor Control using PI Controller implemented on STM32 Nucleo G474RE using PLECS

The program was created in the PLECS environment and implemented on the STM32 Nucleo G474RE microcontroller. It is designed to control a DC motor using a PI controller. The system simulates a closed-loop speed control, comparing the actual motor speed with a reference speed.
